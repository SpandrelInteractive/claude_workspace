# Project Skills

Create two skill directories for your project:

```
.claude/skills/
  {{PROJECT_ID}}-patterns/
    SKILL.md           # Domain entities, architecture, API patterns, testing
  {{PROJECT_ID}}-workflows/
    SKILL.md           # Workflow definitions (new-entity, new-endpoint, etc.)
```

Copy the templates below, replace `{{PROJECT_ID}}` and `{{PROJECT_NAME}}`, and fill in the sections.

See `.claude/docs/skills-guide.md` for the full skills ecosystem reference.
